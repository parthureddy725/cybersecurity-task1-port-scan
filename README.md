# Task 1 - Network Port Scanning

## Objective

Scan local network and identify open ports.

## Tool Used

Nmap

## Steps

1. Installed Nmap
2. Found IP using ipconfig
3. Scanned network using nmap
4. Saved output in result.txt

## Result

Found devices and open ports like 80, 22

## Risk

Open ports can allow attackers to access system

## Conclusion

Learned basic network scanning
